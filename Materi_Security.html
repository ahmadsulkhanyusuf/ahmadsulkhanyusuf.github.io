Web Security Dasar

Melindungi aplikasi Anda dari serangan SQL Injection menggunakan Prepared Statements.

Apa itu SQL Injection?

Teknik serangan di mana penyerang memasukkan perintah SQL berbahaya melalui form input. Jika query kita hanya menggunakan variabel langsung ($id), data tersebut bisa "memanipulasi" logika database.

Solusi: Prepared Statements

Jangan masukkan variabel langsung ke query. Gunakan tanda tanya ? sebagai placeholder.

// 1. Siapkan template query
$stmt = $conn->prepare("SELECT * FROM users WHERE username = ?");

// 2. Bind parameter (hubungkan variabel ke tanda tanya)
// "s" berarti tipe data String
$stmt->bind_param("s", $username);

// 3. Jalankan command
$stmt->execute();

// 4. Ambil hasil
$result = $stmt->get_result();
