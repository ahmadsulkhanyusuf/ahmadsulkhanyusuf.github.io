üìù Apa itu Pseudocode?

Definisi

Pseudocode adalah deskripsi tertulis dari algoritma pemrograman yang menggunakan struktur mirip bahasa pemrograman, tetapi lebih mudah dipahami manusia.

Ciri-ciri Pseudocode:

  • Tidak bergantung pada bahasa pemrograman tertentu
  • Menggunakan bahasa sehari-hari yang terstruktur
  • Fokus pada logika penyelesaian masalah
  • Tidak bisa langsung dijalankan di komputer

Tujuan Pseudocode:

1. Perencanaan Algoritma

Merencanakan algoritma sebelum coding

2. Komunikasi Tim

Memudahkan komunikasi antar developer

3. Dokumentasi

Sebagai dokumentasi algoritma

4. Minimalkan Error

Meminimalisir kesalahan logika

Komponen Utama Pseudocode:

PROGRAM NamaProgram DEKLARASI {variabel, konstanta, tipe data} DESKRIPSI {algoritma utama}

Contoh Sederhana:

PROGRAM Salam DEKLARASI nama : string DESKRIPSI TAMPILKAN "Masukkan nama: " BACA nama TAMPILKAN "Halo, " + nama

Operator dalam Pseudocode:

Jenis Operator Simbol Keterangan
Aritmatika +, -, *, /, DIV, MOD Penjumlahan, pengurangan, perkalian, pembagian, pembagian bulat, modulus
Perbandingan =, <>, <, >, <=, >= Sama dengan, tidak sama dengan, kurang dari, lebih dari, dll
Logika AND, OR, NOT Dan, atau, tidak

üéÆ Struktur Kendali (Control Structures)

1. SELEKSI (Percabangan)

IF-THEN:
IF nilai >= 75 THEN TAMPILKAN "Lulus" ENDIF
IF-THEN-ELSE:
IF suhu > 37 THEN TAMPILKAN "Demam" ELSE TAMPILKAN "Sehat" ENDIF
CASE-OF (Switch Case):
CASE pilihan OF 1 : TAMPILKAN "Anda memilih menu 1" 2 : TAMPILKAN "Anda memilih menu 2" 3 : TAMPILKAN "Anda memilih menu 3" OTHERWISE: TAMPILKAN "Pilihan tidak valid" ENDCASE

2. PERULANGAN (Looping)

FOR Loop:
FOR i ‚Üê 1 TO 5 DO TAMPILKAN "Iterasi ke-" + i ENDFOR
WHILE Loop:
i ‚Üê 1 WHILE i <= 5 DO TAMPILKAN "Nilai i: " + i i ‚Üê i + 1 ENDWHILE
REPEAT-UNTIL:
i ‚Üê 1 REPEAT TAMPILKAN "Nilai i: " + i i ‚Üê i + 1 UNTIL i > 5

Soal 4: Bilangan Prima MENENGAH

Buat pseudocode untuk menentukan apakah suatu bilangan prima atau bukan.

Lihat Jawaban
PROGRAM CekBilanganPrima DEKLARASI n, i : integer isPrima : boolean DESKRIPSI TAMPILKAN "Masukkan bilangan: " BACA n isPrima ‚Üê TRUE IF n <= 1 THEN isPrima ‚Üê FALSE ELSE FOR i ‚Üê 2 TO n DIV 2 DO IF n MOD i = 0 THEN isPrima ‚Üê FALSE ENDIF ENDFOR ENDIF IF isPrima = TRUE THEN TAMPILKAN n + " adalah bilangan PRIMA" ELSE TAMPILKAN n + " BUKAN bilangan prima" ENDIF
