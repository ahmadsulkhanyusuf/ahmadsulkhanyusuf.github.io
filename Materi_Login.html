Sistem Login & Session

Membatasi akses aplikasi agar hanya pengguna terdaftar yang bisa masuk.

1. Persiapan Tabel User

Buat tabel users di database Anda untuk menyimpan kredensial login.

CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE,
    password VARCHAR(255),
    nama_lengkap VARCHAR(100)
);

2. Mengenal Session

Session adalah cara menyimpan informasi pengguna di server agar tetap tersedia saat pindah halaman. Di PHP, kita memulainya dengan:

<?php
session_start();
$_SESSION['username'] = "admin";
?>

3. Cek Login

Letakkan kode ini di bagian paling atas setiap halaman admin (index, tambah, hapus, dll).

<?php
session_start();
if (!isset($_SESSION['login'])) {
    header("Location: login.php");
    exit;
}
?>

Keamanan adalah prioritas!

Pelajari cara mengamankan query login Anda dari serangan SQL Injection di materi berikutnya.
