Fitur Cetak Laporan

Membuat output fisik dari data digital yang ada di aplikasi Anda.

1. Window Print (Browser)

Cara termudah adalah memanggil fungsi cetak bawaan browser menggunakan JavaScript.

<button onclick="window.print()">Cetak Laporan</button>

2. CSS Media Print

Agar tombol atau navigasi tidak ikut tercetak, gunakan CSS `@media print`.

@media print {
    .no-print { 
        display: none !important; 
    }
}

3. Library PDF (Opsional)

Jika ingin file PDF yang lebih rapi dan bisa didownload secara otomatis, Anda bisa mengeksplorasi library seperti:

  • Dompdf: Merubah HTML/CSS menjadi PDF.
  • TCPDF: Library PHP klasik untuk generate PDF.
  • FPDF: Library ringan dan mudah digunakan.
