Pencarian & Pagination

Memudahkan pengguna menemukan data di antara ribuan baris rekaman.

1. Fitur Search (Keyword)

Gunakan operator LIKE dan wildcard % di SQL untuk mencari teks yang mirip.

SELECT * FROM buku WHERE 
judul LIKE '%$keyword%' OR 
penulis LIKE '%$keyword%';

2. Konsep Pagination

Pagination adalah membagi data menjadi beberapa halaman menggunakan LIMIT dan OFFSET.

// Tampilkan 10 data, mulai dari data ke-0
SELECT * FROM buku LIMIT 0, 10;

// Tampilkan 10 data, mulai dari data ke-10 (Halaman 2)
SELECT * FROM buku LIMIT 10, 10;

3. Menghitung Jumlah Halaman

Anda perlu tahu total data untuk menentukan berapa banyak tombol angka halaman yang muncul.

$totalData = mysqli_num_rows(mysqli_query($conn, "SELECT * FROM buku"));
$jumlahHalaman = ceil($totalData / $dataPerHalaman);
