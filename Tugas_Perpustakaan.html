Sistem Informasi Perpustakaan

Tantangan proyek mandiri: Terapkan kemampuan CRUD PHP untuk membangun aplikasi peminjaman buku sederhana.

Deskripsi Proyek

Aplikasi ini bertujuan untuk mendigitalkan proses peminjaman buku di perpustakaan sekolah. Fitur utama mencakup manajemen data buku, anggota, dan pencatatan transaksi peminjaman.

MySQL Database

Penyimpanan data buku & anggota.

PHP Logic

Logika transaksi & validasi data.

Responsive UI

Interface menggunakan CSS/Tailwind.

Instruksi Database

Buatlah database bernama db_perpustakaan dan buatlah tabel-tabel berikut melalui menu "Structure" di phpMyAdmin Anda.

1. Tabel buku

Master Data
Nama Kolom Tipe Data Keterangan
id_buku INT Primary Key, Auto Increment
judul VARCHAR (100) Judul buku
penulis VARCHAR (50) Nama penulis
penerbit VARCHAR (50) Nama penerbit
stok INT Jumlah ketersediaan buku

2. Tabel anggota

Master Data
Nama Kolom Tipe Data Keterangan
id_anggota INT Primary Key, Auto Increment
nama VARCHAR (100) Nama lengkap anggota
alamat TEXT Alamat tinggal
telp VARCHAR (15) Nomor telepon aktif

3. Tabel peminjaman

Transaction
Nama Kolom Tipe Data Relasi / Keterangan
id_pinjam INT Primary Key, Auto Increment
id_buku INT Foreign Key (Hubungkan ke tabel buku)
id_anggota INT Foreign Key (Hubungkan ke tabel anggota)
tgl_pinjam DATE Tanggal mulai pinjam
tgl_kembali DATE Batas tanggal pembalian

Langkah Pengerjaan

Tahap 1: Setup Koneksi

Pastikan PHP terhubung ke database db_perpustakaan melalui file koneksi.

Tahap 2: Manajemen Master

Buat halaman CRUD lengkap untuk data Buku dan Anggota.

Tahap 3: Fitur Transaksi

Buat form peminjaman yang mengambil data dari tabel buku dan anggota (Dropdown).

Tahap 4: Finishing

Tambahkan fitur "Kembalikan Buku" yang akan menghapus data dari tabel peminjaman.

Tantangan Lanjutan (Level Up!)

Sistem Login & Admin

Buat halaman login agar data perpustakaan tidak bisa diakses sembarangan.

Upload Cover Buku

Tambahkan fitur upload gambar untuk menampilkan cover buku di dashboard.

Laporan Detail (JOIN)

Tampilkan Nama Peminjam & Judul Buku secara otomatis di tabel riwayat.

Cetak PDF / Laporan

Buat fitur untuk mencetak riwayat peminjaman ke printer atau PDF.

Tips Sukses Proyek

  • Tulis kode secara bertahap, jangan langsung semuanya. Mulai dari Tampil Data.
  • Gunakan mysqli_error($conn) untuk melacak kesalahan query SQL.
  • Eksperimenlah dengan tampilan (CSS) agar aplikasi terlihat profesional.
