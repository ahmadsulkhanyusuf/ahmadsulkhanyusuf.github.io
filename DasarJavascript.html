1. Pengenalan JavaScript

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi yang awalnya dikembangkan untuk membuat halaman web interaktif. Berbeda dengan HTML (yang mendefinisikan struktur) dan CSS (yang mengatur tampilan), JavaScript memberikan perilaku dinamis pada halaman web.

Contoh Sederhana JavaScript

// Menampilkan pesan popup
alert("Halo, Dunia!");

// Menulis ke konsol browser
console.log("Ini pesan di konsol");

// Mengubah konten HTML
document.getElementById("demo").innerHTML = "Halo JavaScript!";

Sejarah Singkat

JavaScript dibuat oleh Brendan Eich pada tahun 1995 dalam waktu hanya 10 hari! Awalnya bernama Mocha, kemudian diganti menjadi LiveScript, dan akhirnya JavaScript untuk mengambil keuntungan dari popularitas bahasa Java saat itu.

Kenapa Belajar JavaScript?

  • Wajib untuk pengembangan web - Hampir semua website modern menggunakan JavaScript
  • Mudah dipelajari - Sintaks yang relatif mudah dibanding bahasa lain
  • Versatile - Bisa digunakan untuk frontend, backend, mobile apps, desktop apps
  • Komunitas besar - Banyak resources, library, dan framework tersedia
  • Peluang karir - Permintaan tinggi untuk developer JavaScript

Cara Mudah Mengingat: JavaScript vs Java

JavaScript dan Java adalah dua bahasa yang berbeda!

Analogi: JavaScript seperti "kopi" (ringan, untuk web) sedangkan Java seperti "java" (kuat, untuk aplikasi enterprise).

Tips: Ingat bahwa JavaScript berjalan di browser, sedangkan Java perlu di-compile.

2. Variabel & Tipe Data

Apa itu Variabel?

Variabel adalah tempat penyimpanan data di memori yang memiliki nama dan dapat menampung nilai yang dapat berubah selama program berjalan.

var (Cara Lama)

Fungsi-scoped, bisa di redeklarasi

var nama = "Budi";
var nama = "Andi"; // Bisa

let (Modern)

Block-scoped, tidak bisa di redeklarasi

let umur = 16;
// let umur = 17; // Error

const (Konstanta)

Block-scoped, nilai tidak bisa diubah

const PI = 3.14;
// PI = 3.15; // Error

Tipe Data dalam JavaScript

Tipe Data Contoh Keterangan
String "Hello", 'World' Data teks, diapit tanda kutip
Number 25, 3.14, -10 Angka (integer dan float)
Boolean true, false Nilai kebenaran
Array [1, 2, 3], ["a", "b"] Kumpulan data dalam []
Object {nama: "Budi", umur: 16} Data dengan properti dalam {}
Null null Nilai kosong yang disengaja
Undefined undefined Variabel belum diberi nilai

Cara Mudah Mengingat: Tipe Data

String = Seperti string/tali yang menghubungkan karakter

Number = Nomor/number untuk perhitungan

Boolean = Hanya punya 2 nilai, seperti saklar on/off

Array = Aray (deretan) data seperti kereta

Object = Seperti objek di dunia nyata yang punya sifat

3. Operator dalam JavaScript

Jenis-jenis Operator

1. Operator Aritmatika

Operator Contoh Hasil Keterangan
+ 5 + 3 8 Penjumlahan
- 5 - 3 2 Pengurangan
* 5 * 3 15 Perkalian
/ 6 / 3 2 Pembagian
% 7 % 3 1 Modulus (sisa bagi)
** 2 ** 3 8 Eksponensial (pangkat)

2. Operator Perbandingan

console.log(5 == "5"); // true (nilai sama)
console.log(5 === "5"); // false (nilai dan tipe berbeda)
console.log(5 != "5"); // false
console.log(5 !== "5"); // true
console.log(5 > 3); // true
console.log(5 < 3); // false
console.log(5 >= 5); // true
console.log(5 <= 3); // false

3. Operator Logika

let benar = true, salah = false;

console.log(benar && salah); // AND: false
console.log(benar || salah); // OR: true
console.log(!benar); // NOT: false

// Contoh praktis
let umur = 16;
let punyaKTP = false;
console.log(umur >= 17 && punyaKTP); // false

Cara Mudah Mengingat: Operator Perbandingan

== vs ===:

== seperti "kembar identik" - nilai sama saja

=== seperti "kembar identik plus" - nilai dan tipe harus sama

Tips: Selalu gunakan === dan !== untuk menghindari bug!

4. Struktur Kontrol

Percabangan (Conditional Statements)

1. if-else if-else

let nilai = 85;

if (nilai >= 90) {
  console.log("Grade: A");
} else if (nilai >= 80) {
  console.log("Grade: B");
} else if (nilai >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: D");
}
// Output: Grade: B

2. switch-case

let hari = "Senin";
let aktivitas;

switch (hari) {
  case "Senin":
    aktivitas = "Meeting mingguan";
    break;
  case "Jumat":
    aktivitas = "Sholat Jumat";
    break;
  case "Sabtu":
  case "Minggu":
    aktivitas = "Libur";
    break;
  default:
    aktivitas = "Kerja biasa";
}

console.log(aktivitas); // Output: Meeting mingguan

Perulangan (Loops)

1. for loop

// Menampilkan angka 1-5
for (let i = 1; i <= 5; i++) {
  console.log(i);
}

// Menampilkan array
let buah = ["Apel", "Jeruk", "Mangga"];
for (let i = 0; i < buah.length; i++) {
  console.log(buah[i]);
}

2. while loop

// Menampilkan angka 1-5
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}

// Contoh dengan kondisi kompleks
let password = "";
while (password !== "rahasia123") {
  password = prompt("Masukkan password:");
}
console.log("Login berhasil!");

Cara Mudah Mengingat: Perulangan

for = Untuk kasus ketika kita TAHU berapa kali pengulangan

while = Untuk kasus ketika kita TIDAK TAHU berapa kali pengulangan

Tips: Gunakan for untuk array, while untuk input validation

5. Fungsi (Function)

Apa itu Fungsi?

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi dieksekusi ketika "dipanggil". Fungsi membantu dalam:

  • Mengorganisir kode
  • Menghindari pengulangan kode
  • Membuat kode lebih mudah dibaca dan dipelihara

Cara Mendefinisikan Fungsi

Function Declaration

function sapa(nama) {
  return "Halo, " + nama;
}

// Pemanggilan
sapa("Budi"); // "Halo, Budi"

Bisa dipanggil sebelum didefinisikan

Function Expression

const sapa = function(nama) {
  return "Halo, " + nama;
};

// Pemanggilan
sapa("Budi"); // "Halo, Budi"

Harus didefinisikan sebelum dipanggil

Arrow Function

const sapa = (nama) => {
  return "Halo, " + nama;
};

// Atau lebih singkat
const sapa = nama => "Halo, " + nama;

Modern, sintaks lebih pendek

Parameter dan Return Value

// Fungsi dengan parameter
function hitungLuas(panjang, lebar) {
  let luas = panjang * lebar;
  return luas;
}

// Pemanggilan dengan argumen
let hasil = hitungLuas(5, 3);
console.log(hasil); // 15

// Fungsi tanpa return value
function tampilkanPesan(pesan) {
  console.log(pesan);
}
tampilkanPesan("Halo!"); // Menampilkan "Halo!"

Cara Mudah Mengingat: Fungsi

Function seperti mesin kopi:

Parameter = Biji kopi dan air (input)

Function body = Mesin kopi (proses)

Return value = Kopi yang sudah jadi (output)

Pemanggilan = Memasukkan koin dan menekan tombol

6. DOM Manipulation

Apa itu DOM?

DOM (Document Object Model) adalah representasi struktur halaman web sebagai pohon objek yang memungkinkan JavaScript untuk mengakses dan memanipulasi elemen HTML.

DOM adalah...

Interface yang memungkinkan JavaScript mengubah konten, struktur, dan style halaman web

Selektor DOM

Method Contoh Keterangan
getElementById() document.getElementById("judul") Pilih elemen berdasarkan ID
getElementsByClassName() document.getElementsByClassName("item") Pilih elemen berdasarkan class
getElementsByTagName() document.getElementsByTagName("div") Pilih elemen berdasarkan tag
querySelector() document.querySelector("#container .item") Pilih elemen pertama yang cocok
querySelectorAll() document.querySelectorAll(".item") Pilih semua elemen yang cocok

Manipulasi Konten dan Style

let elemen = document.getElementById("demo");

// Mengubah konten
elemen.textContent = "Teks baru";
elemen.innerHTML = "<strong>Teks tebal</strong>";

// Mengubah style
elemen.style.color = "red";
elemen.style.backgroundColor = "yellow";
elemen.style.fontSize = "20px";

// Mengubah atribut
elemen.setAttribute("class", "kelas-baru");
elemen.className = "kelas-lain";

Membuat dan Menghapus Elemen

// Membuat elemen baru
let divBaru = document.createElement("div");
divBaru.textContent = "Ini elemen baru";
divBaru.className = "item-baru";

// Menambahkan ke DOM
document.body.appendChild(divBaru);
// atau
document.getElementById("container").appendChild(divBaru);

// Menghapus elemen
divBaru.remove();
// atau
document.getElementById("container").removeChild(divBaru);

7. Event Handling

Apa itu Event?

Event adalah tindakan atau kejadian yang terjadi di halaman web, seperti klik mouse, penekanan tombol, atau pengiriman form.

Jenis-jenis Event

Kategori Event Keterangan
Mouse Events click Elemen diklik
dblclick Elemen diklik dua kali
mouseover Mouse berada di atas elemen
mouseout Mouse meninggalkan elemen
Keyboard Events keydown Tombol ditekan
keyup Tombol dilepas
keypress Tombol ditekan (deprecated)
Form Events submit Form dikirim
change Nilai input berubah
focus Elemen mendapat fokus
blur Elemen kehilangan fokus

Cara Menambahkan Event Listener

1. Inline HTML Attribute (Tidak Disarankan)

<button onclick="alert('Diklik!')">Klik Saya</button>

2. DOM Property (Cukup Baik)

let tombol = document.getElementById("tombol");
tombol.onclick = function() {
  alert("Tombol diklik!");
};

3. addEventListener (Direkomendasikan)

let tombol = document.getElementById("tombol");
tombol.addEventListener("click", function() {
  alert("Tombol diklik!");
});

// Dengan arrow function
tombol.addEventListener("click", () => {
  alert("Tombol diklik!");
});

Cara Mudah Mengingat: Event Handling

Event seperti bel rumah:

Event Listener = Orang yang mendengar bel

Event Handler = Tindakan yang dilakukan ketika bel berbunyi

Event Object = Informasi tentang bel (siapa yang menekan, kapan, dll)

8. Array & Object

Array

Array adalah struktur data untuk menyimpan kumpulan nilai dalam satu variabel.

// Membuat array
let buah = ["Apel", "Jeruk", "Mangga"];
let angka = [1, 2, 3, 4, 5];
let campuran = [1, "teks", true, null];

// Mengakses elemen array (indeks dimulai dari 0)
console.log(buah[0]); // "Apel"
console.log(buah[1]); // "Jeruk"

// Mengubah elemen array
buah[1] = "Pisang";
console.log(buah); // ["Apel", "Pisang", "Mangga"]

// Panjang array
console.log(buah.length); // 3

Method Array yang Sering Digunakan

let arr = [1, 2, 3];

// Menambah elemen
arr.push(4); // [1, 2, 3, 4] - tambah di akhir
arr.unshift(0); // [0, 1, 2, 3, 4] - tambah di awal

// Menghapus elemen
arr.pop(); // [0, 1, 2, 3] - hapus dari akhir
arr.shift(); // [1, 2, 3] - hapus dari awal

// Method lainnya
arr.includes(2); // true - cek apakah ada
arr.indexOf(2); // 1 - cari posisi elemen
arr.join("-"); // "1-2-3" - gabung jadi string
arr.reverse(); // [3, 2, 1] - balik urutan

Object

Object adalah struktur data yang menyimpan data dalam bentuk pasangan key-value.

// Membuat object
let siswa = {
  nama: "Budi Santoso",
  umur: 16,
  kelas: "10 RPL",
  nilai: {
    matematika: 85,
    bahasa: 90
  },
  aktif: true
};

// Mengakses properti object
console.log(siswa.nama); // "Budi Santoso"
console.log(siswa["umur"]); // 16
console.log(siswa.nilai.matematika); // 85

// Mengubah properti
siswa.umur = 17;
siswa["kelas"] = "11 RPL";

// Menambah properti baru
siswa.hobi = "Membaca";

// Menghapus properti
delete siswa.aktif;

Cara Mudah Mengingat: Array vs Object

Array seperti kereta - setiap gerbong punya nomor (indeks)

Object seperti kartu identitas - setiap info punya label (key)

Tips: Gunakan array untuk data sejenis, object untuk data dengan karakteristik berbeda
