SQL JOIN untuk Kelas 10 RPL

Apa itu JOIN?

JOIN adalah operasi dalam SQL yang digunakan untuk menggabungkan baris dari dua atau lebih tabel berdasarkan kolom terkait antara tabel-tabel tersebut.

Mengapa JOIN penting?

  • Basis data yang baik menggunakan normalisasi (pemecahan tabel)
  • JOIN memungkinkan kita menggabungkan data dari beberapa tabel
  • Menghindari duplikasi data yang tidak perlu
  • Memudahkan maintenance dan update data

Jenis-Jenis JOIN

  1. INNER JOIN
  2. LEFT JOIN (LEFT OUTER JOIN)
  3. RIGHT JOIN (RIGHT OUTER JOIN)
  4. FULL JOIN (FULL OUTER JOIN)
  5. CROSS JOIN

INNER JOIN

Pengertian

INNER JOIN menampilkan hanya baris yang memiliki nilai yang cocok di kedua tabel. Jika tidak ada kecocokan, baris tidak akan ditampilkan.

Sintaks

SELECT kolom1, kolom2, ...
FROM tabel1
INNER JOIN tabel2 ON tabel1.kolom = tabel2.kolom;

LEFT JOIN

Pengertian

LEFT JOIN menampilkan semua baris dari tabel kiri (tabel pertama) dan baris yang cocok dari tabel kanan. Jika tidak ada kecocokan, kolom dari tabel kanan akan berisi NULL.

Sintaks

SELECT kolom1, kolom2, ...
FROM tabel1
LEFT JOIN tabel2 ON tabel1.kolom = tabel2.kolom;

RIGHT JOIN dan FULL JOIN

RIGHT JOIN

RIGHT JOIN menampilkan semua baris dari tabel kanan (tabel kedua) dan baris yang cocok dari tabel kiri. Jika tidak ada kecocokan, kolom dari tabel kiri akan berisi NULL.

SELECT kolom1, kolom2, ...
FROM tabel1
RIGHT JOIN tabel2 ON tabel1.kolom = tabel2.kolom;

FULL JOIN

FULL JOIN menampilkan semua baris ketika ada kecocokan di salah satu tabel. Jika tidak ada kecocokan, kolom dari tabel yang tidak memiliki data akan berisi NULL.

SELECT kolom1, kolom2, ...
FROM tabel1
FULL JOIN tabel2 ON tabel1.kolom = tabel2.kolom;

Contoh Praktis dan Kesimpulan

Contoh Query Praktis

Menampilkan siswa beserta jurusan dan nilai

SELECT s.nama, j.nama_jurusan, n.mata_pelajaran, n.nilai
FROM siswa s
INNER JOIN jurusan j ON s.id_jurusan = j.id_jurusan
LEFT JOIN nilai n ON s.id_siswa = n.id_siswa
WHERE n.nilai > 80
ORDER BY s.nama, n.mata_pelajaran;

Kesimpulan

  • INNER JOIN digunakan ketika kita hanya ingin data yang memiliki hubungan di kedua tabel
  • LEFT JOIN digunakan ketika kita ingin semua data dari tabel utama, meskipun tidak ada hubungan di tabel kedua
  • RIGHT JOIN adalah kebalikan dari LEFT JOIN (jarang digunakan)
  • FULL JOIN digunakan ketika kita ingin semua data dari kedua tabel
  • Pemilihan jenis JOIN tergantung pada kebutuhan query dan hubungan antar tabel

Tips Penting

Selalu gunakan alias untuk mempermudah penulisan

Contoh:

SELECT s.nama, j.nama_jurusan
FROM siswa AS s
INNER JOIN jurusan AS j ON s.id_jurusan = j.id_jurusan;

Perhatikan performa query dengan JOIN

JOIN yang kompleks dapat memperlambat query. Gunakan INDEX pada kolom yang sering di-JOIN untuk meningkatkan performa.

Gunakan WHERE untuk memfilter hasil JOIN

Filter data setelah JOIN dengan klausa WHERE untuk mendapatkan hasil yang spesifik.
