Pengenalan HTML Form

Apa itu HTML Form?

HTML Form adalah elemen HTML yang digunakan untuk mengumpulkan input dari pengguna.

Fungsi HTML Form:

  • Mengumpulkan data dari pengguna
  • Mengirim data ke server untuk diproses
  • Membuat interaksi antara pengguna dan website

Struktur Dasar Form:

<form action="proses.php" method="post"> <!-- Elemen-elemen form di sini --> <input type="submit" value="Kirim"> </form>

Atribut-atribut Penting

Atribut Fungsi
action Menentukan ke mana data form akan dikirim
method Menentukan metode pengiriman data (GET/POST)
name Memberikan nama untuk setiap elemen form
required Membuat field wajib diisi sebelum form dapat dikirim
placeholder Memberikan petunjuk di dalam field input

Perbedaan GET dan POST

GET: Data dikirim melalui URL (terlihat di address bar)

POST: Data dikirim secara tersembunyi (lebih aman)

Validasi Form

Validasi form dapat dilakukan dengan HTML5 untuk memastikan data yang dimasukkan sesuai format yang diharapkan.

<input type="email" name="email" required placeholder="contoh@email.com"> <input type="number" name="usia" min="1" max="100"> <input type="password" name="password" minlength="8">

Jenis-jenis Validasi:

  • required: Field harus diisi
  • min/max: Batas nilai untuk angka
  • minlength/maxlength: Batas panjang teks
  • pattern: Pola regex untuk validasi khusus
  • type="email": Memastikan format email

Latihan Praktikum

Tugas 1: Form Kontak Sederhana

Buatlah form kontak dengan field:

  • Nama (wajib diisi)
  • Email (wajib diisi, format email)
  • Subjek
  • Pesan (textarea)

Tugas 2: Form Pendaftaran Event

Buatlah form pendaftaran event dengan field:

  • Nama lengkap
  • Institusi/sekolah
  • Jenis kelamin (radio button)
  • Kategori event (dropdown)
  • Makanan vegetarian (checkbox)
  • Komentar (textarea)

Kriteria Penilaian:

  • Struktur HTML yang benar
  • Penggunaan elemen form yang tepat
  • Penerapan atribut yang sesuai
  • Validasi form yang efektif
