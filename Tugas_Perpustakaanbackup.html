Sistem Informasi Perpustakaan

Tantangan proyek mandiri: Terapkan kemampuan CRUD PHP untuk membangun aplikasi peminjaman buku sederhana.

Deskripsi Proyek

Aplikasi ini bertujuan untuk mendigitalkan proses peminjaman buku di perpustakaan sekolah. Fitur utama mencakup manajemen data buku, anggota, dan pencatatan transaksi peminjaman.

MySQL Database

Penyimpanan data buku & anggota.

PHP Logic

Logika transaksi & validasi data.

Responsive UI

Interface menggunakan CSS/Tailwind.

Rencana Database

Gunakan SQL berikut untuk membuat struktur database di phpMyAdmin Anda.

1. Tabel Buku

CREATE TABLE buku (
    id_buku INT PRIMARY KEY AUTO_INCREMENT,
    judul VARCHAR(100),
    penulis VARCHAR(50),
    penerbit VARCHAR(50),
    stok INT
);

2. Tabel Anggota

CREATE TABLE anggota (
    id_anggota INT PRIMARY KEY AUTO_INCREMENT,
    nama VARCHAR(100),
    alamat TEXT,
    telp VARCHAR(15)
);

3. Tabel Peminjaman

CREATE TABLE peminjaman (
    id_pinjam INT PRIMARY KEY AUTO_INCREMENT,
    id_buku INT,
    id_anggota INT,
    tgl_pinjam DATE,
    tgl_kembali DATE,
    FOREIGN KEY (id_buku) REFERENCES buku(id_buku),
    FOREIGN KEY (id_anggota) REFERENCES anggota(id_anggota)
);

Langkah Pengerjaan

Tahap 1: Setup Koneksi

Pastikan PHP terhubung ke database db_perpustakaan melalui file koneksi.

Tahap 2: Manajemen Master

Buat halaman CRUD lengkap untuk data Buku dan Anggota.

Tahap 3: Fitur Transaksi

Buat form peminjaman yang mengambil data dari tabel buku dan anggota (Dropdown).

Tahap 4: Finishing

Tambahkan fitur "Kembalikan Buku" yang akan menghapus data dari tabel peminjaman.

Tips Sukses Proyek

  • Tulis kode secara bertahap, jangan langsung semuanya. Mulai dari Tampil Data.
  • Gunakan mysqli_error($conn) untuk melacak kesalahan query SQL.
  • Eksperimenlah dengan tampilan (CSS) agar aplikasi terlihat profesional.
