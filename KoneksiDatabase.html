Koneksi PHP ke MySQL

Langkah demi langkah menyambungkan PHP dengan Database secara runut

01 / 10

Kenapa Harus Connect?

Agar aplikasi web kita bisa menyimpan data secara permanen, kita harus menghubungkan skrip PHP dengan sistem database seperti MySQL.

Analogi Mudahnya

Bayangkan PHP itu seperti otak yang bisa berpikir dan membuat keputusan, sedangkan MySQL itu seperti lemari arsip yang menyimpan semua data penting.

Koneksi = Jembatan antara otak dan lemari arsip. Tanpa jembatan ini, otak tidak bisa menyimpan atau mengambil data dari lemari!

Simpan Data

Registrasi, transaksi, postingan.

Mirip mencatat di buku agenda

Ambil Data

Mencari dan menampilkan informasi.

Seperti membuka halaman tertentu

Update Data

Mengubah profil atau stok barang.

Lupa mencoret-coret di buku

1

Persiapan Lingkungan

Pastikan XAMPP (atau paket serupa) sudah terinstall dan berjalan di komputer kamu.

Aktifkan Service

  • Klik tombol Start pada Apache.
  • Klik tombol Start pada MySQL.

XAMPP = Apache + MySQL + PHP + Perl
(Semudah instal game PC!)

2

Buat Database di phpMyAdmin

  1. a

    Buka browser dan ketik http://localhost/phpmyadmin

  2. b

    Pilih menu "New" atau "Baru" di sisi kiri.

  3. c

    Masukkan nama database, contoh: db_belajar

  4. d

    Klik tombol Create atau Buat.

Ingat nama database ini, karena kita akan menggunakannya di kode PHP nanti.

3

Menyiapkan Variabel Koneksi

Buat file baru bernama koneksi.php dan definisikan parameter berikut:

$servername = "localhost";

Alamat server (biasanya localhost)

$username = "root";

Default user XAMPP adalah root

$password = "";

Default password XAMPP adalah kosong

$dbname = "db_belajar";

Nama database yang dibuat tadi

4

Gunakan mysqli_connect()

Inilah fungsi utama untuk "berjabat tangan" dengan MySQL.

<?php
// Membuat koneksi
$conn = mysqli_connect($servername, $username, $password, $dbname);
?>
Hasil dari fungsi ini akan disimpan dalam variabel $conn. Variabel inilah yang akan kita panggil setiap kali ingin berinteraksi dengan database.
5

Cek Status Koneksi

Kita harus memastikan koneksi berhasil sebelum lanjut eksekusi perintah lain.

<?php
// Mengecek koneksi
if (!$conn) {
    die("Koneksi Gagal: " . mysqli_connect_error());
}
echo "Koneksi Berhasil!";
?>

Fungsi die()

Menghentikan program seketika dan menampilkan pesan error.

Ini seperti memutuskan untuk tidak lanjut jika gagal!

mysqli_connect_error()

Menyediakan alasan detail kenapa koneksi gagal.

seperti GPS error yang menunjukkan lokasi masalah!

Penting untuk Dipahami!

Kenapa harus cek koneksi?

Sama seperti mengecek apakah kabel printer tercolok sebelum mencetak. Jika tidak tercolok, pasti gagal!

Variabel $conn itu apa?

Ini seperti remote control TV. Setelah konek, remote ($conn) inilah yang kita gunakan untuk mengontrol database.

6

Menutup Koneksi

"Jangan biarkan pintu terbuka jika sudah tidak digunakan."

Maksudnya: Setelah selesai ngobrol dengan database, tutup baik-baik koneksinya.

mysqli_close($conn);

Menutup koneksi akan mengosongkan sumber daya server yang digunakan.

Wawasan Penting

Secara teknis, PHP akan menutup koneksi secara otomatis saat script selesai dijalankan. Namun, menutupnya secara manual adalah praktik coding yang bersih (clean code) dan sangat direkomendasikan untuk aplikasi besar.

Contoh Kode Utuh

koneksi.php 
<?php
$host     = "localhost";
$user     = "root";
$password = "";
$db       = "db_belajar";

// Membuat koneksi
$conn = mysqli_connect($host, $user, $password, $db);

// Cek Koneksi
if (!$conn) {
    die("Koneksi Gagal: " . mysqli_connect_error());
}

echo "Koneksi Berhasil!";
?>

Error yang Sering Muncul

"Access denied for user..."

Biasanya karena username salah (default: root) atau password yang tidak sesuai.

Mirip pakai kunci yang salah

"Unknown database..."

Nama database di PHP tidak sama dengan yang ada di phpMyAdmin. Typo!

Periksa huruf besar/kecil

"Could not connect to server"

MySQL di XAMPP belum di-Start. Cek Control Panel XAMPP.

"Fatal error: Uncaught Error..."

Ada kesalahan penulisan (syntax) di file PHP kamu. Cek titik koma.

Kode tidak sesuai aturan

Tips Debugging untuk Pemula

Cek Langkah demi Langkah:

  1. 1. Apache & MySQL sudah Start?
  2. 2. Database sudah dibuat?
  3. 3. Nama database sama persis?
  4. 4. Username & password benar?

Trik Cepat:

  • • Copy-paste nama database dari phpMyAdmin
  • • Gunakan default: root & password kosong
  • • Cek baris yang error di kode
  • • Restart XAMPP jika masih error
