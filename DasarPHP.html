BAB I

Pengenalan PHP

Slide 1/8

1.1 Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk pengembangan web dinamis.

Karakteristik PHP

  • Dijalankan di server, bukan browser
  • Ekstensi file: .php
  • Dapat disisipkan dalam HTML

Sejarah Singkat

  • Dibuat oleh Rasmus Lerdorf (1994)
  • Versi terbaru: PHP 8.x
  • Open Source & Gratis

Contoh Kode PHP Sederhana

<?php
    // Ini adalah komentar
    echo "Halo, Dunia!";
    print "Saya belajar PHP";
?>

Keunggulan PHP

Mudah Dipelajari

Kompatibel dengan Database

Komunitas Besar

Cross Platform

3.1 Sintaks Dasar PHP 

<?php

Aturan Dasar:

  • Kode PHP diawali dengan <?php

  • Setiap statement diakhiri titik koma (;)

Komentar tidak dieksekusi oleh PHP, digunakan untuk dokumentasi kode.

3.3 Variabel dalam PHP

Variabel di PHP diawali dengan tanda $ dan bersifat case-sensitive. 

<?php
    // Deklarasi variabel
    $nama = "Andi";
    $umur = 16;
    $tinggi = 165.5;
    $status = true;
    
    // Menampilkan variabel
    echo "Nama: " . $nama;  // Concatenation
    echo "Umur: $umur tahun";  // Interpolasi string
?>

Slide 4/8

4.1 Operator Aritmatika 

<?php
    $a = 10;
    $b = 3;
    
    // Operator Aritmatika
    echo $a + $b;  // Penjumlahan: 13

    echo $a * $b;  // Perkalian: 30
    echo $a / $b;  // Pembagian: 3.333
    echo $a % $b;  // Modulus: 1
    echo $a ** $b; // Pangkat: 1000
?>

4.2 Operator Perbandingan 

<?php
    $x = 10;
    $y = "10";
    
    var_dump($x == $y);   // true (nilai sama)
    var_dump($x === $y);  // false (tipe berbeda)
    var_dump($x != $y);   // false
    var_dump($x > $y);    // false
?>

Perbedaan == dan ===

== membandingkan nilai saja (loose comparison)

=== membandingkan nilai DAN tipe data (strict comparison)

4.3 Operator Logika 

<?php
    $benar = true;
    $salah = false;
    
    var_dump($benar && $salah); // AND: false
    var_dump($benar || $salah); // OR: true
    var_dump(!$benar);          // NOT: false
?>
&&

AND

Keduanya harus true

||

OR

Salah satu true

!

NOT

Kebalikan nilai

5.2 Switch-Case 

<?php
    $hari = "Senin";
    
    switch($hari) {
        case "Senin":
            echo "Hari pertama sekolah";
            break;
        case "Sabtu":
            echo "Hari libur";
            break;
        default:
            echo "Hari aktif";
    }
?>

Penting!

Jangan lupa menambahkan break setelah setiap case. Jika tidak, eksekusi akan berlanjut ke case berikutnya!

5.3 Perulangan (Looping)

FOR Loop

for($i=1; $i<=5; $i++) {
    echo "Iterasi ke-$i";
}

Digunakan ketika jumlah iterasi diketahui

WHILE Loop

$j = 1;
while($j <= 3) {
    echo "While loop $j";
    $j++;
}

Digunakan ketika jumlah iterasi tidak pasti

DO-WHILE Loop

$k = 1;
do {
    echo "Do-while $k";
    $k++;
} while($k <= 3);

Minimal 1x eksekusi meski kondisi false

6.2 Fungsi di PHP 

<?php
    // Fungsi tanpa parameter
    function salam() {
        return "Selamat pagi!";
    }
    
    // Fungsi dengan parameter
    function sapa($nama) {
        return "Halo, $nama!";
    }
    
    // Fungsi dengan nilai default
    function kenalan($nama, $kota = "Jakarta") {
        return "Saya $nama dari $kota";
    }
    
    // Memanggil fungsi
    echo salam();  // Output: Selamat pagi!
    echo sapa("Budi");  // Output: Halo, Budi!
?>

Manfaat Fungsi

  • Reusability

    Kode dapat digunakan berulang kali

  • Modularitas

    Memecah program menjadi bagian kecil

  • Maintainability

    Mudah diperbaiki dan dikembangkan

Sintaks Fungsi:
function namaFungsi($parameter) {
  // kode fungsi
  return $nilai;
}

Slide 7/8

7.1 Form HTML dengan PHP

File: form.html

<!DOCTYPE html>
<html>
<head>
    <title>Form Input</title>
</head>
<body>
    <form action="proses.php" method="POST">
        <h2>Form Pendaftaran</h2>
        
        <label>Nama:</label><br>
        <input type="text" name="nama"><br><br>
        
        <label>Email:</label><br>
        <input type="email" name="email"><br><br>
        
        <input type="submit" value="Kirim">
    </form>
</body>
</html>

Atribut Form Penting:

action

Menentukan file yang akan memproses form

action="proses.php"
method

Metode pengiriman data: GET atau POST

GET POST
Perbedaan GET vs POST

GET: Data terlihat di URL, untuk data tidak sensitif

POST: Data tidak terlihat di URL, untuk data sensitif

7.2 Memproses Form dengan PHP

File: proses.php

<?php
    // Mengambil data dari form
    $nama = $_POST['nama'];
    $email = $_POST['email'];
    
    // Validasi sederhana
    if(empty($nama) || empty($email)) {
        echo "<div class='error'>Semua field harus diisi!</div>";
    } else {
        // Proses data (simpan ke database, dll)
        echo "<h3>Terima kasih <span class='highlight'>$nama</span>!</h3>";
        echo "<p>Email <strong>$email</strong> berhasil terdaftar.</p>";
    }
?>
$_POST

Superglobal untuk mengambil data dari method POST

$_GET

Superglobal untuk mengambil data dari method GET

empty()

Fungsi untuk mengecek apakah variabel kosong

Keamanan Form

Selalu validasi dan sanitasi input pengguna untuk mencegah serangan:

SQL Injection

Gunakan mysqli_real_escape_string() atau prepared statements

XSS (Cross-Site Scripting)

Gunakan htmlspecialchars() untuk output

PROYEK AKHIR

Sistem Login Sederhana

Slide 8/8

Tugas Akhir Semester

Buat sistem login sederhana menggunakan PHP dan MySQL yang menerapkan konsep-konsep yang telah dipelajari.

Struktur Database

-- Buat database
CREATE DATABASE sekolah;

-- Buat tabel users
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    password VARCHAR(255) NOT NULL,
    nama_lengkap VARCHAR(100),
    level ENUM('admin', 'siswa') DEFAULT 'siswa'
);

-- Insert data contoh
INSERT INTO users (username, password, nama_lengkap, level)
VALUES ('admin', '21232f297a57a5a743894a0e4a801fc3', 'Administrator', 'admin');

File: login.php

<?php
    session_start();
    $koneksi = mysqli_connect(
        "localhost", 
        "root", 
        "", 
        "sekolah"
    );
    
    if(isset($_POST['login'])) {
        $username = $_POST['username'];
        $password = md5($_POST['password']);
        
        $query = "SELECT * FROM users WHERE 
                 username='$username' AND 
                 password='$password'";
        
        $result = mysqli_query($koneksi, $query);
        
        if(mysqli_num_rows($result) == 1) {
            $_SESSION['username'] = $username;
            header("Location: dashboard.php");
        } else {
            echo "Login gagal! Username atau password salah.";
        }
    }
?>

Fitur yang Harus Ada

Login System

Form login dengan validasi

Session Management

Menggunakan session untuk login state

CRUD Operations

Create, Read, Update, Delete data

Logout System

Fitur logout dengan session destroy
