Apa itu CRUD?

CRUD adalah singkatan dari operasi dasar yang dilakukan pada data di database. Hampir semua aplikasi web di dunia menggunakan konsep ini.

CRUD dalam Kehidupan Sehari-hari

Bayangkan kamu punya buku telepon untuk menyimpan nomor teman-teman:

C - CREATE: Menulis nomor teman baru
R - READ: Mencari nomor teman yang sudah ada
U - UPDATE: Mengubah nomor teman yang pindah
D - DELETE: Menghapus nomor teman yang tidak lagi dihubungi
C

CREATE

Menambah data baru

R

READ

Membaca/Menampilkan data

U

UPDATE

Mengubah data lama

D

DELETE

Menghapus data

Langkah 1: Siapkan Tabel Database

Buatlah tabel bernama siswa di database kamu:

CREATE TABLE siswa (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nama VARCHAR(100),
    kelas VARCHAR(20),
    alamat TEXT
);

ID (PK)

Sebagai pengenal unik dan dibuat otomatis.

Mirip NIS/nomor absen yang unik

VARCHAR

Untuk menyimpan teks pendek (Nama/Kelas).

Seperti kolom di buku absen

TEXT

Untuk menyimpan teks yang panjang (Alamat).

Seperti catatan panjang di jurnal

Langkah 2: Menampilkan Data (READ)

Buat file index.php untuk menampilkan data dalam tabel HTML:

<?php
include 'koneksi.php';
$query = mysqli_query($conn, "SELECT * FROM siswa");
?>

<!-- Tombol Tambah Data -->
<a href="tambah.php" 
   class="btn-tambah">
    <i class="fas fa-plus"></i> Tambah Data
</a>

<table>
    <tr>
        <th>Nama</th>
        <th>Kelas</th>
        <th>Aksi</th>
    </tr>
    <?php while($data = mysqli_fetch_array($query)): ?>
    <tr>
        <td><?= $data['nama'] ?></td>
        <td><?= $data['kelas'] ?></td>
        <td>
            <a href="edit.php?id=<?= $data['id'] ?>">Edit</a>
            <a href="hapus.php?id=<?= $data['id'] ?>">Hapus</a>
        </td>
    </tr>
    <?php endwhile; ?>
</table>
Tombol Tambah Data

Tombol ini mengarah ke halaman tambah.php untuk menambahkan data baru. Letakkan tombol ini di atas tabel agar mudah ditemukan user.

Penjelasan Kode (Bahasa Gampang)

  • mysqli_query menjalankan perintah SELECT untuk mengambil semua data siswa.

    Mirip meminta tolong ke penjaga perpustakaan: "Cariin semua buku siswa ya!"

  • while loop digunakan untuk mengulang baris tabel selama data masih ada.

    Seperti membuka satu per satu halaman buku sampai habis

  • mysqli_fetch_array memecah hasil query menjadi array yang mudah dibaca.

    Mengubah data mentah menjadi informasi terstruktur seperti daftar isi

Langkah 3: Menambah Data (CREATE)

Analogi: Seperti mengisi formulir pendaftaran sekolah dan menyerahkannya ke TU untuk disimpan dalam arsip.

1

Buat Form HTML

Siapkan halaman tambah.php dengan form input

2

Tangkap Data POST

Ambil data dari form menggunakan $_POST

3

Simpan ke Database

Gunakan query INSERT INTO untuk menyimpan

STEP 1

Form Input (tambah.php)

<!DOCTYPE html>
<html>
<head>
    <title>Tambah Data Siswa</title>
</head>
<body>
    <h2>Form Tambah Siswa</h2>
    
    <form action="proses_tambah.php" 
          method="POST">
        
        <label>Nama Siswa:</label>
        <input type="text" 
               name="nama" 
               required>
        <br><br>
        
        <label>Kelas:</label>
        <input type="text" 
               name="kelas" 
               required>
        <br><br>
        
        <label>Alamat:</label>
        <textarea name="alamat" 
                  required></textarea>
        <br><br>
        
        <button type="submit">Simpan</button>
    </form>
</body>
</html>

Penting: Atribut name harus sama dengan nama kolom di database!

STEP 2 & 3

Proses Simpan (proses_tambah.php)

<?php
// STEP 2: Hubungkan ke database
include 'koneksi.php';

// STEP 2: Tangkap data dari form
$nama   = $_POST['nama'];
$kelas  = $_POST['kelas'];
$alamat = $_POST['alamat'];

// STEP 3: Buat query INSERT
$sql = "INSERT INTO siswa 
        (nama, kelas, alamat) 
        VALUES 
        ('$nama', '$kelas', '$alamat')";

// Jalankan query dan cek keberhasilan
if (mysqli_query($conn, $sql)) {
    // Berhasil, kembali ke halaman index
    header("location:index.php");
} else {
    // Gagal, tampilkan error
    echo "Error: " . mysqli_error($conn);
}
?>

Tips Keamanan

  • Selalu validasi input sebelum simpan ke database
  • Gunakan mysqli_real_escape_string() untuk mencegah SQL Injection
  • Atau lebih baik gunakan Prepared Statement

Langkah 4: Mengubah Data (UPDATE)

Konsep UPDATE: Operasi ini memiliki 2 TAHAP

TAHAP 1 Ambil Data Lama

Mengisi form dengan data yang sudah ada

TAHAP 2 Simpan Perubahan

Update data baru ke database

Analogi: Seperti mengedit profil di Instagram. Pertama kamu lihat profil lama, baru edit dan klik "Simpan Perubahan"

1
Tangkap ID

Ambil ID dari URL menggunakan $_GET

2
Query SELECT

Ambil data berdasarkan ID tersebut

3
Tampilkan Form

Isi form dengan data lama di atribut value

4
Update Data

Simpan perubahan dengan query UPDATE

FILE 1

Form Edit (edit.php)

<?php
// STEP 1: Tangkap ID dari URL
include 'koneksi.php';
$id = $_GET['id'];

// STEP 2: Ambil data siswa berdasarkan ID
$query = mysqli_query($conn, 
    "SELECT * FROM siswa WHERE id='$id'");
$data = mysqli_fetch_array($query);
?>

<!-- STEP 3: Tampilkan form dengan data lama -->
<form action="proses_edit.php" 
      method="POST">
    
    <!-- Kirim ID lewat hidden input -->
    <input type="hidden" 
           name="id" 
           value="<?= $data['id'] ?>">
    
    <label>Nama:</label>
    <input type="text" 
           name="nama" 
           value="<?= $data['nama'] ?>">
    <br><br>
    
    <label>Kelas:</label>
    <input type="text" 
           name="kelas" 
           value="<?= $data['kelas'] ?>">
    <br><br>
    
    <label>Alamat:</label>
    <textarea name="alamat"><?= $data['alamat'] ?></textarea>
    <br><br>
    
    <button type="submit">Update</button>
</form>

Penting: Gunakan hidden input untuk mengirim ID ke proses_edit.php

FILE 2

Proses Update (proses_edit.php)

<?php
include 'koneksi.php';

// Tangkap semua data dari form
$id     = $_POST['id'];
$nama   = $_POST['nama'];
$kelas  = $_POST['kelas'];
$alamat = $_POST['alamat'];

// Query UPDATE dengan WHERE
$sql = "UPDATE siswa SET 
        nama   = '$nama', 
        kelas  = '$kelas', 
        alamat = '$alamat' 
        WHERE id = '$id'";

// Jalankan query
if (mysqli_query($conn, $sql)) {
    header("location:index.php");
} else {
    echo "Error: " . mysqli_error($conn);
}
?>

PERINGATAN PENTING!

‚ùå JANGAN LUPA WHERE id='$id'

Tanpa WHERE, SEMUA DATA di tabel akan berubah menjadi nilai yang sama!

WHERE berfungsi seperti "alamat rumah" yang spesifik menunjuk data mana yang akan diubah

Langkah 5: Menghapus Data (DELETE)

PERHATIAN: Ini adalah operasi TERMUDAH namun PALING BERBAHAYA!

Analogi: Seperti menekan tombol DELETE FILE di komputer. Sekali terhapus, data hilang selamanya (kecuali ada backup)!

1

Tangkap ID

Ambil ID data yang akan dihapus dari URL

2

Eksekusi DELETE

Jalankan query DELETE dengan WHERE clause

3

Redirect

Kembali ke halaman utama setelah berhasil

File: hapus.php

DANGER ZONE 
<?php
// STEP 1: Hubungkan ke database
include 'koneksi.php';

// STEP 2: Tangkap ID dari URL
$id = $_GET['id'];

// STEP 3: Buat query DELETE
$sql = "DELETE FROM siswa 
        WHERE id='$id'";

// STEP 4: Eksekusi query
if (mysqli_query($conn, $sql)) {
    // Berhasil, redirect ke index
    header("location:index.php");
} else {
    // Gagal
    echo "Error: " . mysqli_error($conn);
}
?>
Struktur Query DELETE
DELETE FROM nama_tabel WHERE kondisi

WHERE sangat penting! Tanpa WHERE, semua data akan terhapus

Tambahkan Konfirmasi di index.php 

<!-- Di bagian tombol hapus -->
<a href="hapus.php?id=<?= $data['id'] ?>"
   onclick="return confirm('Yakin hapus data ini?')">
    Hapus
</a>

Fungsi confirm() akan menampilkan popup konfirmasi. Jika user klik "Cancel", data tidak akan dihapus.

Tips Keamanan DELETE

WAJIB Selalu gunakan WHERE clause

Tanpa WHERE, semua record di tabel akan terhapus!

RECOMMENDED Tambahkan konfirmasi

Gunakan JavaScript confirm() atau modal konfirmasi

PRO TIP Soft Delete

Di aplikasi profesional, data tidak dihapus permanen. Tambahkan kolom is_deleted atau deleted_at

Bonus: Soft Delete

Ubah status menjadi "terhapus" tanpa menghapus data:

UPDATE siswa SET is_deleted=1 WHERE id='$id'
